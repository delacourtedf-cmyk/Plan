📖 Notice d'utilisation
Installation & chargement des vidéos
1 — Héberger sur GitHub Pages
1
Créer un compte GitHub
Rendez-vous sur github.com et créez un compte gratuit (si ce n'est pas déjà fait).
2
Créer un nouveau dépôt
Cliquez sur le + en haut à droite → New repository. Nom : par exemple setteriq. Cochez Public et Add a README file, puis cliquez Create repository.
3
Déposer le fichier HTML
Dans le dépôt, cliquez Add file → Upload files. Renommez le fichier en index.html avant de le glisser, puis cliquez Commit changes.
4
Activer GitHub Pages
Allez dans Settings → Pages. Sous "Source", choisissez Deploy from a branch, branch main, dossier / (root) puis Save.
5
Récupérer l'adresse
Après 1-2 min, votre site est en ligne à : https://VOTRE-PSEUDO.github.io/setteriq/
2 — Installer sur iPad
1
Ouvrir dans Safari
Sur l'iPad, ouvrez Safari et entrez l'adresse de votre site GitHub Pages.
2
Ajouter à l'écran d'accueil
Tapez l'icône de partage ↑ → Sur l'écran d'accueil. L'app se lancera en plein écran avec l'icône SetterIQ.
3 — Charger les vidéos
Méthode A — ZIP (recommandé sur iPad)
1
Créer le ZIP sur PC
Sur Windows, clic droit sur votre dossier principal (contenant les sous-dossiers Freeze, Live, Continu) → Envoyer vers → Dossier compressé.
2
Transférer sur iPad
Copiez le .zip sur iCloud Drive, Google Drive, ou via une clé USB + adaptateur.
3
Importer dans SetterIQ
Dans Admin Staff → Créer un pack, cliquez Importer le ZIP du pack. Les sous-dossiers sont détectés automatiquement.
Méthode B — Dossier par dossier (sur ordinateur)
1
Charger Freeze
Cliquez Sélectionner un dossier → choisissez le dossier Freeze.
2
Charger Live puis Continu
Recliquez sur le même bouton → choisissez Live, puis recommencez pour Continu. Les imports s'accumulent automatiquement.
3
Vérifier et sauvegarder
Vérifiez que le compteur affiche 170/170 (60 Freeze + 60 Live + 50 Continu), puis cliquez Enregistrer le pack.
